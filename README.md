# projectstrip

这个shell脚本支持将工程中无用的图片资源文件识别删除，支持iOS，Android，另外支持iOS assets图片资源集中资源的识别，简单易用轻巧.


使用方法：
show unused image file

sh strip.sh -p /path/of/your/project
show and clean unused image file

sh strip.sh -r -p /path/of/your/project
